# 뷰로 만든 투두리스트

```powerShell
  # json-server
  yarn server

  # project setup
  yarn install

  # compiles and hot-reloads for development
  # http://localhost:8077
  yarn serve
```

### 가능한 기능

- 투두아이템 등록
- 투두아이템 수정
- 투두아이템 삭제
- 전체 목록 삭제
- 선택하여 여러 개 삭제
- 선택하여 여러 개 동시에 완료 처리

### 수정하고 싶은 것

- 선택하여 여러 개 아이템 업데이트 한 이후의 화면 리렌더링 이슈
